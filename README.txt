stx-python3
-----------

This repository enables starlinxg developers to use
python2.7/python3.6/python3.9 to test their code better
using tox and common python developer tools.

Note: Must have docker installed and configured.

Build
-----
docker build -t python3 .

Run
---

docker run -i -t python3

at charles.short@windriver.com.
